## How to Update/Use
Either:  
1. Download from the release (for stable release)  

2. Download the entire code in zip and replace over existing  

3. Git pull if you have git installed:  
   - Navigate to the PTCGPB Bot folder 
   - Add remote if needed: `git remote add upstream https://github.com/josephcy95/PTCGPB.git`  
   - Fetch and merge: `git pull upstream main`
